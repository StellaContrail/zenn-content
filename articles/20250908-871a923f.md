## はじめに
`az deployment` コマンドを実行した際に、エラーメッセージ "The content for this response was already consumed" が表示されてデプロイできない場合がある。具体的なスタックトレースも表示されず、何をしてよいかわからない。

```bash
$ az deployment group create --resource-group test-rg --template-file infra/main.json
The content for this response was already consumed
```

## 対処法

`--debug` オプションをつけて実行する

```bash
$ az deployment group create --resource-group <リソースグループ名> --template-file <ファイル名> --debug
```
